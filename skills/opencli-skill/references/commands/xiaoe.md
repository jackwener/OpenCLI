# xiaoe

## Commands

### catalog
- Purpose: 小鹅通课程目录（支持普通课程、专栏、大专栏）
- Args: None
- Usage: `opencli xiaoe catalog [options] -f json`

### content
- Purpose: 提取小鹅通图文页面内容为文本
- Args: None
- Usage: `opencli xiaoe content [options] -f json`

### courses
- Purpose: 列出已购小鹅通课程（含 URL 和店铺名）
- Args: None
- Usage: `opencli xiaoe courses [options] -f json`

### detail
- Purpose: 小鹅通课程详情（名称、价格、学员数、店铺）
- Args: None
- Usage: `opencli xiaoe detail [options] -f json`

### play-url
- Purpose: 小鹅通视频/音频/直播回放 M3U8 播放地址
- Args: None
- Usage: `opencli xiaoe play-url [options] -f json`
